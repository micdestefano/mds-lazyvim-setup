# mds-lazyvim-setup
Setup script for LazyVim with the configuration by Michele De Stefano

# Warning

- This setup is for Linux/Unix OS only.
- When the script runs it erases your `~/.config/nvim`, `~/.cache/nvim` and
  `~/.local/{share,state}/nvim` directories.

# Usage

- Clone the repository into a folder that is not into `~/.config`, `~/.cache` or
  `~/.local`.
- Run the `mds-lazyvim-setup.sh` script as a normal user.

After these two steps the `~/.config/nvim` directory is configured on the `mds-lazyvim`
branch and it is ready to be used. The `mds-lazyvim` branch is the one where I make
my customizations.
